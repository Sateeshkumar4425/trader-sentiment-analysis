# Trader Performance vs Market Sentiment Analysis

## Project Overview
This project analyzes how Bitcoin market sentiment (Fear & Greed Index) influences trader behavior and trading performance.

The goal is to identify behavioral patterns and derive actionable insights for trading strategies.

## Dataset
The project uses two datasets:

1. Trade execution dataset (Hyperliquid trading records)
2. Bitcoin Fear & Greed sentiment dataset

## Technologies Used
Python
Pandas
NumPy
Matplotlib
Seaborn
Jupyter Notebook

## Key Steps
1. Data cleaning and preprocessing
2. Sentiment classification (Fear / Greed / Neutral)
3. Merging trade data with sentiment data
4. Behavioral analysis of traders
5. Performance metrics evaluation

## Key Insights
- Traders increase long exposure during greed sentiment
- Fear regimes show higher volatility and downside risk
- Frequent traders tend to be more sentiment sensitive

## Conclusion
Incorporating sentiment-aware risk management strategies can improve trading stability and risk-adjusted returns.
